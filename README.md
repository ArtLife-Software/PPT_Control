![PowerPoint](https://img.shields.io/badge/Target-PowerPoint-B7472A?logo=microsoftpowerpoint&logoColor=white)

## 🚀 主要功能
控制 PowerPoint 演示。使用鍵盤或簡報筆時，PowerPoint **不須取得焦點**（可在後台控制）。

* **控制按鍵**：`PageUp`, `PageDown`, `Up`, `Down`, `Left`, `Right`
* **運作邏輯**：
    * 當 PowerPoint 演示時，此六按鍵只傳送至 PowerPoint，其他程式無法使用。
    * 當 PowerPoint 停止演示，此六按鍵恢復原本功能。

---

## 💻 操作方式
可在 PowerPoint 演示前或演示後，執行 PPT Control。

---

## ⚠️ 注意事項
為了確保 COM 元件運作正常，請根據您的 Office 版本選擇對應位元：
* **PowerPoint x86 (32位元)**：須執行 `PPT Control x86.exe`
* **PowerPoint x64 (64位元)**：須執行 `PPT Control x64.exe`

> **提示**：如果您不確定版本，可以先嘗試 x64，若無法運作再更換為 x86。
